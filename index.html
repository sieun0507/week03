아두이노 기반 기억력 게임: 사이먼 게임

회로 설계 및 코드 분석

발표자: 7조 남지연, 오시은

날짜: 3주차 2차시

1. 프로젝트 소개

'사이먼 게임'은 빛과 소리의 패턴을 기억하고 따라 하는 고전적인 기억력 게임입니다. 이 프로젝트는 바로 이 '사이먼 게임'을 아두이노로 구현한 것으로, 사용자는 아두이노가 제시하는 무작위 순서를 기억하고, 올바른 버튼을 눌러 순서를 재현해야 합니다.

주요 목표

  • 사용자에게 무작위 순서 제시 (LED 점등 및 부저 소리).
  • 사용자 입력 확인 및 정오 판별.
  • 성공 또는 실패 시 명확한 피드백 제공.
페이지 1/5

2. 하드웨어 구성

이 게임은 다음의 주요 부품들로 구성됩니다.

  • 아두이노 우노: 모든 부품을 제어하는 핵심 마이크로컨트롤러입니다.
  • 4색 LED (빨강, 노랑, 초록, 파랑): 시각적 순서를 나타냅니다.
  • 버튼 4개: 사용자가 순서를 입력하는 데 사용됩니다.
  • 부저: 각 음표와 성공/실패 효과음을 출력합니다.

회로 연결은 코드 내의 `LED_PINS`, `BUTTON_PINS`, `BUZZER_PIN` 변수에 따라 구성되었습니다.

페이지 2/5

3. 소프트웨어 및 게임 로직

게임 상태 (GameState)

  • READY: 초기화 상태.
  • INTRO: 게임 시작 전 알림 시퀀스.
  • PLAYING: 게임 진행 중, 순서 제시 및 입력 대기.
  • SUCCESS: 정답 시 상태.
  • FAIL: 오답 또는 시간 초과 시 상태.

핵심 함수

  • randomizeSequence(): 게임마다 새로운 무작위 순서 생성.
  • playRandomSequence(): 생성된 순서에 따라 LED와 부저를 동작.
  • checkUserInput(): 사용자가 누른 버튼과 시스템이 제시한 순서를 비교.
페이지 3/5

4. 게임 플레이 및 결과

게임 진행

  • 시작: 모든 LED가 켜졌다가 꺼진 후, 부저와 LED를 통해 시작 알림 시퀀스가 재생됩니다.
  • 순서 제시: 무작위로 생성된 3개의 순서(색상, 소리)가 간헐적으로 재생됩니다.
  • 사용자 입력: 사용자는 제시된 순서에 맞춰 버튼을 누릅니다.
  • 게임 기회: 각 순서에 대한 입력 기회는 1회이며, 순서가 틀리거나 20초 안에 입력하지 못하면 게임이 즉시 종료됩니다.

결과

  • 성공: 모든 순서를 정확히 맞추면 '반짝반짝 작은별' 멜로디가 연주되고 모든 LED가 점등됩니다.
  • 실패: 순서가 틀리거나 시간 초과 시, 모든 LED가 3회 깜박이며 실패를 알립니다.
페이지 4/5

5. 요약 및 질의응답

이 프로젝트는 아두이노와 간단한 부품들을 활용하여 구현한 기억력 게임으로, 하드웨어와 소프트웨어를 통합적으로 이해하는 데 도움이 됩니다.

개선 및 확장 방안

  • 게임 난이도를 조절하여 더 긴 순서나 빠른 재생 속도를 적용할 수 있습니다.
  • LCD 스크린을 추가하여 점수나 현재 상태를 시각적으로 보여줄 수 있습니다.
  • 멀티플레이어 기능을 추가하여 친구들과 함께 즐길 수 있도록 확장할 수 있습니다.

질문 있으시면 자유롭게 말씀해주세요.
